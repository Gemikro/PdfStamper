    Sequence
    Name Type/Ref Min Max Description
    pdf_template_list template 1 1 One or more pdf templates to fill. Resulting file is merged pdf. 
    data xs:string 1 1 Data to inject into pdf editable form. 
    read_only xs:boolean 1 1 Should we flatten editable form after editing ? . 
    debug_mode xs:boolean 1 1 Is debug mode enabled. Verbose logging and mapping information is written to log. 
    merge_output xs:boolean 1 1 If several templates are rendered, should we merge the output or not? 
    output_root_path xs:string 1 1 Client can request to get the Unc path of the rendered response. If empty, base64 encoded content is returned
